## Welcome to My Todo List

A full-stack todo application built with React, TypeScript, and Express. This project demonstrates my skills in both frontend and backend development, featuring user authentication, task management, and a clean, responsive interface.

### Features

- User Authentication with Firebase
- Create, read, update, and delete todos
- Mark todos as complete/incomplete
- Clean and intuitive user interface
- RESTful API for data persistence
- Secure user authentication with JWT
- Account management (registration, deletion)

### Technologies Used

Frontend:
- React 18
- TypeScript
- Styled Components
- React Router
- Firebase Authentication
- Axios for API calls

Backend:
- Node.js with Express
- TypeScript
- Prisma ORM
- JWT Authentication
- Firebase Admin SDK
- RESTful API Architecture
